## Da backrooms Script
An open source GUI Da Backrooms script.

> [!NOTE]
> Mobile may behave differently than PC.

> [!IMPORTANT]
> Down below is a list of all the features the script currently has.

Main Tab
> Walkspeed Adjuster  
> Infinite Stamina  
> Infinite Sanity
>
Combat Tab
> Auto Shoot   
> Auto Spam Ability     
> Disable Sickle Puddles    
> Disable Waves   
> Disable Portals   
> Custom No Item Loss   
> 
Duplication
> Inventory Duplication     
> Dupe Others    
> Dev Dupe   
> Prevent missing textures (Client)
> 
Visual
> Entity ESP
> Disable Camera Effects
> No Blur
> Show Vault          

Experimental
> Replace bone scythes animation with the trident  
> Item Aura
> Proximity Prompts Range increase and shorten hold duration
> Show Aegis Ring
> Brick Dupe
> Auto Consume
>
Hub
> Clear inventory
> Spin Button
> Auto Spin
