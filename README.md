## Da backrooms Script
An open source GUI Da Backrooms script.

> [!NOTE]
> Mobile may behave differently than PC.

> [!IMPORTANT]
> Down below is a list of all the features the script currently has.

Main Tab
> Walkspeed Adjuster.  
> Infinite Stamina.  
> Infinite Sanity.
>
Combat Tab
> Auto Shoot.   
> Auto Spam Ability.     
> Disable Sickle Puddles.    
> Disable Waves.   
> Disable Portals.   
> Custom No Item Loss.
